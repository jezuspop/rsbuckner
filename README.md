# Richard Buckner - Personal Landing Page

A personal linktree-style landing page for Richard Buckner, Christian Pastor + Digital Evangelist.

## Structure

```
├── index.html      (Main landing page)
├── church.html     (My Local Church page)
├── media.html      (My Media Ministry page)
└── README.md
```

## Pages

- **index.html** - Main landing page with links to:
  - My Local Church
  - My Family of Churches
  - My Media Ministry
  - Your Prayer Request

- **church.html** - Local church information page with:
  - Church details (Montgomery County Congregation, Greater Philadelphia Church of Christ)
  - Map Location button
  - Visit Website button

- **media.html** - Media ministry page with:
  - Podcast button
  - JezusPOP! button

## Setup

1. Clone or download this repository
2. Upload all files to your GitHub repository
3. Enable GitHub Pages in repository Settings → Pages
4. Set source to **main** branch and **/ (root)** folder
5. Your site will be live at `https://yourusername.github.io/repository-name/`

## Customization

- Update links in HTML files to point to your actual URLs
- Modify colors in the `:root` CSS variables
- Adjust animation speeds in the CSS
